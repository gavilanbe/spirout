# 🧚 Spirout
Prototipo 3D del protagonista de *Spirout*: un pequeño duende del bosque de cabeza amarilla, antena en espiral y collar de hojas, modelado con estética Nintendo 64.

## ✨ Características
- Personaje completo modelado de forma 100% procedural con primitivas de Three.js (~350 polígonos).
- Antena espiral generada con `TubeGeometry` sobre una curva Catmull-Rom.
- Collar de hojas con 16 conos dispuestos en doble anillo.
- Detalles dibujados con texturas de canvas (espirales en mejillas, panza y mano).
- Cabeza facetada con `IcosahedronGeometry` para el look low-poly retro.
- Cero instalación: Three.js vía CDN.

## 🚀 Cómo jugar / ejecutar
```bash
open index.html
```
Funciona en cualquier navegador moderno, sin dependencias ni build.

## 🎮 Controles
- Arrastra: girar la cámara.
- Scroll: zoom.
- Doble-click: reset de cámara.

## 🛠️ Tecnología
- JavaScript + Three.js (vía CDN).
- `OrbitControls` para la cámara.
- Modelado procedural con primitivas y texturas de canvas.
